# 🟣🎮 Retro Friv-Style Portfolio — Manel

## 🧠 Core Concept
A playful, nostalgic, and bold portfolio inspired by **old Friv games**, but reimagined with your **robotics + AI personality**.

> Think: **early 2000s browser games + neon violet aesthetic + interactive playground**

This is NOT a classic portfolio.  
It’s a **clickable game-like interface** where each section feels like entering a mini-world.

---

## 🎨 Visual Identity

### 🎯 Style Direction
- Inspired by old **Friv.com interfaces**
- Grid-based clickable icons (like a game selection screen)
- Bright, saturated colors with a **dominant violet theme**
- Slightly chaotic but FUN and intentional
- Cartoonish + tech fusion

---

### 🎨 Color Palette
- **Primary:** #6A0DAD (Deep Violet)
- **Secondary:** #9B5DE5 (Bright Purple)
- **Accent 1:** #00F5D4 (Cyan pop)
- **Accent 2:** #F15BB5 (Pink neon)
- **Background:** Gradient violet → dark purple
- **Hover glow:** Soft neon outlines

---

### 🔤 Typography
- Headings: **Arcade / Pixel fonts** (Press Start 2P, Bungee)
- Body: **Rounded playful font** (Poppins / Comic Neue for fun touch)

---

## 🧩 Website Structure (Friv Grid Style)

---

### 1. 🎮 Home Screen (Main Interface)

🧠 This is the KEY idea:
- Full screen = **grid of square icons**
- Each icon = a section of your portfolio
- Like old Friv games selection

🟪 Example Icons:
- 🤖 Robot → Projects
- 👩‍💻 Avatar → About Me
- ⚡ Plug → Smart Plug Project
- ✈️ Plane → Aerobotix / RC builds
- 🧠 Brain → AI Work
- 📊 Graph → Skills
- 📡 Antenna → Contact

✨ Hover effect:
- Icon grows + glows
- Small animation (bounce, spin, pulse)

---

### 2. 👩‍💻 About Me (Mini Game Page)

Style:
- Looks like a **character intro screen**

Include:
- Stylized avatar or photo
- Short bio in a **fun tone**
- Stats like a game character:


NAME: Manel
CLASS: Engineer
LEVEL: 23
SKILLS: Robotics / AI / Embedded Systems
SPECIAL ABILITY: Builds things from scratch


---

### 3. 🤖 Projects (Game Levels)

Each project = **a game tile or level**

Click → opens a **mini environment**

Examples:
- Robot project → small animated robot scene
- AI project → moving nodes / brain animation
- Smart plug → glowing circuit board style

Each includes:
- Description
- Tech stack
- Your role
- Demo (video/GIF)

---

### 4. 🧪 Playground (Interactive Zone)

Make it fun like a mini-game:
- Click buttons to trigger:
  - Robot movement animation
  - AI visualization
- Or simple “experiment” UI

💡 Reflects your personality:
> learning = playing

---

### 5. 🖼️ Gallery (Memory Arcade)

Style:
- Tiles like game thumbnails
- Slight randomness like Friv

Content:
- Events (Aerobotix)
- Builds
- Sports
- Daily life

✨ Effects:
- Hover = zoom + glow
- Click = full screen pop-up

---

### 6. 📊 Skills (Upgrade Screen)

Style:
- Like a **skill tree or upgrade panel**

Example:

[ROBOTICS] █████████░ 90%
[PYTHON] ██████████ 95%
[STM32] ████████░░ 80%
[AI] █████████░ 85%


Or:
- Clickable nodes unlocking skills

---

### 7. 📡 Contact (Final Level)

Style:
- Looks like a **mission complete / communication screen**

Text:
> “Ready to collaborate?”

Button:
`[ SEND MESSAGE ]`

Fun idea:
- Button says: **“START MISSION”**

- Don’t overload with animations
- Keep loading fast
- Mobile = stacked grid version
